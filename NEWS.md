# News file for changes on GLib Mock

This file documents the changes on each release.

These are ordered from latest to oldest.

## 0.1.0 (2026-04-09)

### Added:

- First release! Fully functional mocking framework, with functions for adding mocks, getting real implementations, and of course boilerplate functions for initialization and committing mock functions.
- Various examples in tests/ directory on both UNIX-like (Linux/macOS) and Win32 platforms.
- Fully compatible with Meson build system, the project can be added as a subproject.
- Fully open-source, open-to-collabarations.
- GLib-style documentation [here](https://otaxhu.github.io/glib-mock)
